## 프로젝트 생성하기
```
mkdir 디렉토리이름
npx create-react-app 프로젝트이름
cd 디렉토리이름/프로젝트이름
npm start
```

그냥 설치하면 최상위 폴더의 하위 폴더로 설치되므로 연습용 폴더를 만든 다음 설치하는 것을 추천함.

바벨이랑 웹팩이 모두 설치되어 있으므로 신경쓸 것이 없어
